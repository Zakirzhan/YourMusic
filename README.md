# YourMusic

I developed a small, simple Music App to try AppleMusic API, "Clean Swift" architecture and hybrid layout with UIKit and SwiftUI
- UIKit/xib/SwiftUI
- Clean Swift
- AVKit, Alamofire, SDWebImage, UserDefaults, URLImage (for SwiftUI)
